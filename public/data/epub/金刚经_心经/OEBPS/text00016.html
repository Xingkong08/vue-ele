第 dì 九 jiǔ 品 pǐn 一 yī 相 xiàng 无 wú 相 xiàng 分 fèn

须 xū 菩 pú 提 tí 。 于 yú 意 yì 云 yún 何 hé 。 须 xū 陀 tuó 洹 huán 能 néng 作 zuò 是 shì 念 niàn 。 我 wǒ 得 dé 须 xū 陀 tuó 洹 huán 果 guǒ 不 fǒu 。 须 xū 菩 pú 提 tí 言 yán 。 不 bú 也 yě 。 世 shì 尊 zūn 。 何 hé 以 yǐ 故 gù 。 须 xū 陀 tuó 洹 huán 名 míng 为 wéi 入 rù 流 liú 。 而 ér 无 wú 所 suǒ 入 rù 。 不 bú 入 rù 色 sè 声 shēng 香 xiāng 味 wèi 触 chù 法 fǎ 。 是 shì 名 míng 须 xū 陀 tuó 洹 huán ， 须 xū 菩 pú 提 tí 。 于 yú 意 yì 云 yún 何 hé 。 斯 sī 陀 tuó 含 hán 能 néng 作 zuò 是 shì 念 niàn 。 我 wǒ 得 dé 斯 sī 陀 tuó 含 hán 果 guǒ 不 fǒu 。 须 xū 菩 pú 提 tí 言 yán 。 不 bú 也 yě 。 世 shì 尊 zūn 。 何 hé 以 yǐ 故 gù 。 斯 sī 陀 tuó 含 hán 名 míng 一 yī 往 wǎng 来 lái 。 而 ér 实 shí 无 wú 往 wǎng 来 lái 。 是 shì 名 míng 斯 sī 陀 tuó 含 hán 。 须 xū 菩 pú 提 tí 。 于 yú 意 yì 云 yún 何 hé 。 阿 ā 那 nà 含 hán 能 néng 作 zuò 是 shì 念 niàn 。 我 wǒ 得 dé 阿 ā 那 nà 含 hán 果 guǒ 不 fǒu 。 须 xū 菩 pú 提 tí 言 yán 。 不 bú 也 yě 。 世 shì 尊 zūn 。 何 hé 以 yǐ 故 gù 。 阿 ā 那 nà 含 hán 名 míng 为 wéi 不 bú 来 lái ， 而 ér 实 shí 无 wú 不 bú 来 lái 。 是 shì 名 míng 阿 ā 那 nà 含 hán 。 须 xū 菩 pú 提 tí 。 于 yú 意 yì 云 yún 何 hé 。 阿 ā 罗 luó 汉 hàn 能 néng 作 zuò 是 shì 念 niàn 。 我 wǒ 得 dé 阿 ā 罗 luó 汉 hàn 道 dào 不 fǒu 。 须 xū 菩 pú 提 tí 言 yán 。 不 bú 也 yě 。 世 shì 尊 zūn 。 何 hé 以 yǐ 故 gù 。 实 shí 无 wú 有 yǒu 法 fǎ 名 míng 阿 ā 罗 luó 汉 hàn 。 世 shì 尊 zūn 。 若 ruò 阿 ā 罗 luó 汉 hàn 作 zuò 是 shì 念 niàn 。 我 wǒ 得 dé 阿 ā 罗 luó 汉 hàn 道 dào 。 即 jí 著 zhe 我 wǒ 人 rén 众 zhòng 生 shēng 寿 shòu 者 zhě 。 世 shì 尊 zūn 。 佛 fó 说 shuō 我 wǒ 得 dé 无 wú 诤 zhèng 三 sān 昧 mèi 。 人 rén 中 zhōng 最 zuì 为 wéi 第 dì 一 yī 。 是 shì 第 dì 一 yī 离 lí 欲 yù 阿 ā 罗 luó 汉 hàn 。 我 wǒ 不 bú 作 zuò 是 shì 念 niàn 。 我 wǒ 是 shì 离 lí 欲 yù 阿 ā 罗 luó 汉 hàn 。 世 shì 尊 zūn 。 我 wǒ 若 ruò 作 zuò 是 shì 念 niàn 。 我 wǒ 得 dé 阿 ā 罗 luó 汉 hàn 道 dào 。 世 shì 尊 zūn 则 zé 不 bú 说 shuō 须 xū 菩 pú 提 tí 。 是 shì 乐 lè 阿 ā 兰 lán 那 nà 行 héng 者 zhě 。 以 yǐ 须 xū 菩 pú 提 tí 实 shí 无 wú 所 suǒ 行 xíng 。 而 ér 名 míng 须 xū 菩 pú 提 tí 。 是 shì 乐 yào 阿 ā 兰 lán 那 nà 行 héng 。
