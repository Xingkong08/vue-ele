第 dì 十 shí 品 pǐn 庄 zhuāng 严 yán 净 jìng 土 tǔ 分 fèn

佛 fó 告 gào 须 xū 菩 pú 提 tí 。 于 yú 意 yì 云 yún 何 hé 。 如 rú 来 lái 昔 xī 在 zài 然 rán 灯 dēng 佛 fó 所 suǒ 。 于 yú 法 fǎ 有 yǒu 所 suǒ 得 dé 不 fǒu 。 不 bú 也 yě 。 世 shì 尊 zūn 。 如 rú 来 lái 在 zài 然 rán 灯 dēng 佛 fó 所 suǒ 。 于 yú 法 fǎ 实 shí 无 wú 所 suǒ 得 dé 。 须 xū 菩 pú 提 tí 。 于 yú 意 yì 云 yún 何 hé 。 菩 pú 萨 sà 庄 zhuāng 严 yán 佛 fó 土 tǔ 不 fǒu 。 不 bú 也 yě 。 世 shì 尊 zūn 。 何 hé 以 yǐ 故 gù 。 庄 zhuāng 严 yán 佛 fó 土 tǔ 者 zhě 。 即 jí 非 fēi 庄 zhuāng 严 yán 。 是 shì 名 míng 庄 zhuāng 严 yán 。 是 shì 故 gù 须 xū 菩 pú 提 tí 。 诸 zhū 菩 pú 萨 sà 摩 mó 诃 hē 萨 sà 。 应 yīng 如 rú 是 shì 生 shēng 清 qīng 净 jìng 心 xīn 。 不 bú 应 yīng 住 zhù 色 sè 生 shēng 心 xīn 。 不 bú 应 yīng 住 zhù 声 shēng 香 xiāng 味 wèi 触 chù 法 fǎ 生 shēng 心 xīn 。 应 yīng 无 wú 所 suǒ 住 zhù 而 ér 生 shēng 其 qí 心 xīn 。 须 xū 菩 pú 提 tí 。 譬 pì 如 rú 有 yǒu 人 rén 。 身 shēn 如 rú 须 xū 弥 mí 山 shān 王 wáng ， 于 yú 意 yì 云 yún 何 hé 。 是 shì 身 shēn 为 wéi 大 dà 不 fǒu 。 须 xū 菩 pú 提 tí 言 yán 。 甚 shèn 大 dà 。 世 shì 尊 zūn 。 何 hé 以 yǐ 故 gù 。 佛 fó 说 shuō 非 fēi 身 shēn 。 是 shì 名 míng 大 dà 身 shēn 。
